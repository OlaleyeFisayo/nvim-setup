#### Make a backup of your current nvim and shared folder

```shell

mv ~/.local/state/nvim ~/.local/state/nvim.bak

```

#### Clone the repository

```shell
git clone https://github.com/OlaleyeFisayo/nvim-setup.git ~/.config/nvim
```

#### Start Neovim

```shell
nvim
```
